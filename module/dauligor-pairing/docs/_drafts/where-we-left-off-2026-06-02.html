How the Character Creator works now

Tab: Create (the wheel) Tab: Character (review)

The wheel (Create tab)

  • Center button → Ability Scores. Two modes: Point Buy (32-point budget, 8–16) and a shared world roll pool — every player rolls one 4d6-drop-lowest set into a public pool, anyone can pick any set, the DM can enter sets by hand.
  • Wedges → Class / Species / Background open searchable pickers (Species = the 2024 label over the existing race data).
  • Class shows the rich preview; on Build it hands off to the existing class importer at level 1 for skill/option choices.

The Character tab + Build

  • A review card per section with an Edit jump back to its wedge.
  • Build Character (footer) writes ability scores, embeds the species & background items, then opens the class builder.
  • Opened from a sheet → applies onto that actor; opened from the Actor directory → makes a fresh character.
