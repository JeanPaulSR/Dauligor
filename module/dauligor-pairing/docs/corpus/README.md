# Corpus Folder

Store side-by-side research artifacts here.

Recommended pattern:

- one source-side file
- one Plutonium-imported Foundry dump
- one manually created or official Foundry dump

Suggested filenames:

- `5etools-spell-acid-splash.json`
- `plutonium-item-acid-splash.json`
- `foundry-item-acid-splash.json`
- `5etools-class-fighter.json`
- `plutonium-item-fighter-class.json`
- `foundry-item-fighter-class.json`

This folder is intended for:

- schema research
- regression testing
- helping future agents compare input-side and target-side document shapes
