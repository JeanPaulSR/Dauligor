Settings — 5 Layout Concepts

The shell was a placeholder; the wide Appearance builder broke it. Five ways forward.
docs/_drafts · branch: settings-pages
The driving constraint Most sections (Profile, Security, Maintenance) are narrow forms. Appearance is wide — a controls column plus a live preview pane. The current single narrow content column can't hold both, so it reads cramped. A good layout has to serve both shapes without feeling lopsided. Each concept below is rated on that "Appearance fit".
