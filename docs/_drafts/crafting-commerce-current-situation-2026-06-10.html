Decisions locked (so we don't re-litigate)

  • New tables are camelCase — Foundry is camelCase end-to-end; ordering field is sort. These tables skip the legacy compendium.ts alias layer.
  • Materials extend items; enchantments get their own table. A material is a carryable thing (loot row); an enchantment is a reusable effect applied to many bases.
  • Recipes are the universal spine — one creation rule for items, enchantments, and enchanted items. Structure first; live crafting execution is a later phase.
  • The shop is multiple named shops, not one global price list — a DM can stock specific shops (decided 2026-06-10).
